Ethnicity

To explore whether gaze effects on imitative and spatial compatibility are conditional to the ethnicity of the observed agent, we ran following model:

Condensed form:
  RT ~ Imitative*Gaze*Ethnicity + Spatial*Gaze*Ethnicity + (1|ID) + (1|Stimulus)

Extended form:
  RT ~ Imitative + Spatial + Gaze + Ethnicity + Imitative:Gaze + Spatial:Gaze + 
    Imitative:Ethnicity + Spatial:Ethnicity + Gaze:Ethnicity + Imitative:Gaze:Ethnicity +
    Spatial:Gaze:Ethnicity + (1|ID) + (1|Stimulus)

Effect coding contrasts were set for all categorical predictors. An inverse gaussian distribution with identity link function was selected for this GLMM as we used RTs as dependent variable.

Main results

Besides the main effects of Imitative, Spatial and Gaze, the GLMM revealed a significant 3-way interaction between Imitative:Gaze:Ethnicity, which suggests that the influence of gaze on imitative compatibility is dependent on the ethnicity of the actor.

Follow-up tests

To disentangle the 3-way interaction, we tested the interaction Imitative:Gaze for Mediterranean and Northen-European actors separately. This model showed that gaze had a significant effect on imitative compatibility for Northern-European actors, but not for Mediterranean actors.

Planned comparisons confirmed that Mediterranean actors triggered imitative compatibility effects in both direct gaze and averted gaze conditions. In contrast, Northern-European identities elicited imitative effects only when they established direct gaze, but not with averted gaze.

Social anxiety

To explore whether gaze effects on imitative and spatial compatibility are modulated by individual differences in social anxiety, we ran one GLMM for each of the three social anxiety questionnaires (SIAS, SPS and GARS). The general form of the models was:

Condensed form: RT ~ Imitative*Gaze*Quest + Spatial*Gaze*Quest + (1|ID) + (1|Stimulus)

Extended form: RT ~ Imitative + Spatial + Quest + Gaze + Imitative:Gaze + Spatial:Gaze + Imitative:Quest + Spatial:Quest + Gaze:Quest + Imitative:Gaze:Quest + Spatial:Gaze:Quest + (1|ID) + (1|Stimulus)

Quest refers to the scores of either the SIAS, SPS or GARS, respectively. Effect coding contrasts were set for all categorical predictors and continues predictors were mean-centered. An inverse gaussian distribution with identity link function was selected for this GLMM as we used RTs as dependent variable.

Follow-up tests

To follow-up the 3-way interaction, we tested the conditional effects of SIAS on imitative compatibility for direct gaze and averted gaze conditions separately.

Results from this GLMM indicated that the difference in RTs between imitatively incompatible and imitatively compatible trials (i.e. imitative effect) in averted gaze conditions increased as a function of social anxiety. Although descriptively the imitative effects following direct seemed to decrease with higher social anxiety symptoms, this effect did not reach statistical significance.

Attributed meaning of gaze

After completing the automatic imitation task, participants rated on a 5-point scale (0 = not at all, 4 = a lot) 8 items how they had generally felt when watching the actors directing the gaze towards them (direct gaze) and away from them (averted gaze). The items were the following:

  • I felt observed by the person.
  • I felt connected with the person.
  • I felt accepted by the person.
  • I felt distracted by the person.
  • I felt pressured by the person.
  • I felt ignored by the person.
  • I felt relieved.

A paired-samples t-test was conducted for each item to compare ratings for direct and averted gaze conditions. The Holm–Bonferroni method was used to control for multiple comparisons.

Participants reported feeling significantly more observed, connected with, accepted, distracted and pressured by the targets when they were directed the gaze towards them as compared to away from them. Conversely, averted gaze videos induced more feelings of being ignored and rejected by the target than direct gaze videos. No significant differences were found between gaze conditions with respect to feeling relieved.

# define variables to compare
gaze1 <- c('direct')
gaze2 <- c('averted')
feeling <- c('observed','connected', 'accepted', 'distracted', 'pressured', 'ignored', 'rejected','relieved')
gaze.ttest.pairs <- data.frame(feeling, gaze1, gaze2)

# number of pairwise comparisons
ntests <- nrow(gaze.ttest.pairs)

# create data frame for post-hoc tests results
gaze.ttest <- data.frame(matrix(ncol=12, nrow=ntests))
colnames(gaze.ttest) <- c('feeling', 'm_direct', 'sd_direct', 'm_averted', 'sd_averted', 't', 'df', 'ci_lower', 'ci_upper', 'd', 'p', 'padjust')

# loop though each pairwise comparison
for (t in 1:ntests) {

  gaze.ttest$feeling[t] <- as.character(gaze.ttest.pairs$feeling[t])

  var1 <- subset(d.gaze$rating, d.gaze$gaze=='direct' & d.gaze$feeling==gaze.ttest$feeling[t])
  var2 <- subset(d.gaze$rating, d.gaze$gaze=='averted' & d.gaze$feeling==gaze.ttest$feeling[t])

  gaze.ttest$m_direct[t]   <- mean(var1)
  gaze.ttest$sd_direct[t]  <- sd(var1)
  gaze.ttest$m_averted[t]  <- mean(var2)
  gaze.ttest$sd_averted[t] <- sd(var2)

  gaze.ttest$t[t]  <- t.test(var1, var2, paired=TRUE)$statistic
  gaze.ttest$df[t] <- t.test(var1, var2, paired=TRUE)$parameter
  gaze.ttest$p[t]  <- t.test(var1, var2, paired=TRUE)$p.value
  gaze.ttest$ci_lower[t] <- t.test(var1, var2, paired=TRUE)$conf.int[1]
  gaze.ttest$ci_upper[t] <- t.test(var1, var2, paired=TRUE)$conf.int[2]

  gaze.ttest$d[t] <- lsr::cohensD(var1, var2, method='paired')

}

# apply multiple comparisons adjustment to p-values
gaze.ttest$padjust <- p.adjust(gaze.ttest$p, method='holm')
